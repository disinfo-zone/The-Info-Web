# Psychics & Remote Viewers

Individuals known for psychic abilities, remote viewing, or participation in psi research as test subjects.

```dataview
TABLE WITHOUT ID
  file.link AS Name,
  summary AS Description
FROM "10 - PEOPLE"
WHERE category = "Psychics & Remote Viewers"
SORT file.name ASC
```
